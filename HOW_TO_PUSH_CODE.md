# 🚀 HOW TO PUSH CODE - Quick Guide

## Option 1: Using Git Command Line (Terminal)

```bash
# Step 1: Make sure you're in your project folder
cd your-project-folder

# Step 2: Check what files changed
git status

# Step 3: Add all changes
git add .

# Step 4: Commit with a message
git commit -m "Add new feature" 

# Step 5: Push to GitHub
git push origin main
```

**That's it! Your code is pushed!** 🎉

---

## Option 2: Using GitHub Desktop (Easy GUI)

1. **Open GitHub Desktop**
2. **Select your repository**
3. **Click "Current Branch"**
4. **Make sure you're on "main"**
5. **Click "Fetch origin"** to sync
6. **Make your code changes**
7. **In left panel, you'll see changed files**
8. **Enter commit message** (e.g., "Add new feature")
9. **Click "Commit to main"**
10. **Click "Push origin"** 
11. **Done!** 🚀

---

## Option 3: Using VS Code (Visual Studio Code)

1. **Open your project in VS Code**
2. **Make your code changes**
3. **Click Source Control icon** (left sidebar)
4. **You'll see changed files listed**
5. **Type commit message** in box at top
6. **Click the checkmark** (Commit)
7. **Click "Sync Changes"** or arrow icon
8. **Done!** 🚀

---

## Option 4: Using GitHub Web Interface

1. **Go to https://github.com/corkumandrew557-tech**
2. **Select your repository**
3. **Click "Create new file"** or edit existing
4. **Make changes**
5. **Scroll down**
6. **Click "Commit changes"**
7. **Add commit message**
8. **Select "Commit directly to main branch"**
9. **Click "Commit changes"**
10. **Done!** 🚀
